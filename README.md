## Client Setup

To set up and run the client application:

1. Navigate to the client directory:
```bash
cd client
```

2. Install dependencies:
```bash
npm i
```

3. Start the development server:
```bash
npm run dev
```

## Backend Setup

To set up and run the backend server:

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm i
```

3. Start the server:
```bash
npm start
```
